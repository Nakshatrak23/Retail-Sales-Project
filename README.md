# Retail Sales Data Analysis Project

This project analyzes retail sales data using SQL and Python.

## Repository Contents
- Data Cleaning and Preparation.ipynb — Data merging, cleaning, and feature creation  
- Final_Report.ipynb — Analysis, visualization, and reporting  
- Phase1 Sql.sql — Database schema and data loading for transactions and response tables  

## Project Workflow
1. Data collection from Kaggle  
2. SQL database setup and table creation  
3. Data cleaning and preparation 
4. Data analysis and visualization  
5. Final reporting and insights  

## Tools Used
SQL, Python (Pandas, Matplotlib, Seaborn)
